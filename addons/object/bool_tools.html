Bool Tool¶

待补充。

激活¶

  • 打开Blender，打开用户偏好设置下的插件栏。

  • 选择 物体 然后选择 Bool Tool 以启用插件。

描述¶

概念(Concepts)

  • Brush：是作为布尔运算的蒙版物体。

  • Canvas：是保留布尔运算的物体。

你可以选择任意数量的物体，并将所有这些物体作为一个布尔笔刷应用到活动的物体上。只需选择想要成为画笔的物体，选择一个物体将其设置为活动的物体，然后点击这些操作中的一个即可：

自动布尔式(Auto Boolean)
差集(Difference) Shift-Ctrl-小键盘减

从活动物体中移除物体。

并集(Union) Shift-Ctrl-小键盘加

将选定的物体添加到活动物体中。

交集(Intersection) Shift-Ctrl-小键盘星号

在所选物体与活动物体之间应用交叉操作。

分割(Slice) Shift-Ctrl-小键盘斜杠

待补充。

笔刷布尔(Brush Boolean)
差集(Difference) Ctrl-小键盘减

对物体直接应用差集。

并集(Union) Ctrl-小键盘加

直接对一个物体应用并集。

交集(Intersection) Ctrl-小键盘星号

对物体直接应用交集。

分割(Slice) Ctrl-小键盘斜杠

待补充。

首选(Preferences)¶

快速转化

当在插件首选项中启用这个选项时，你的 G / R / S 快捷键将被替换成一个自定义的快捷键，它可以在变换操作前后处理物体的可见性和布尔修改器，以便在使用布尔操作时提供快速的变换。它只在处理高多边形笔刷的时候效果很好，如果你尝试在低多边形笔刷中使用它，当另一个高多边形笔刷被应用时，它的速度会很慢，因为我们对这种情况的依赖图处理得不好。
