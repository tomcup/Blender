Node Wrangler¶

这个插件为你提供了几个工具，帮助你快速有效地处理节点。许多功能可以同时适用于合成和着色器节点，有些功能还可以将合成中的功能带到着色器节点中。有些工具是为特定的功能而设计的，但是我们的目标是让你在任何情况下都能完全灵活地使用这些工具，即使是我们不打算使用的地方。

激活¶

  • 打开Blender，打开用户偏好设置下的插件栏。

  • 选择 Node 然后选择 Node Wrangler 以启用插件。

使用方法(Usage)¶

使用节点编辑器侧边栏中的面板或 Ctrl-Spacebar 键盘快捷键或单独的工具快捷键。

描述¶

懒人连接(Lazy Connect)¶

参考

快捷键(Hotkey)

Ctrl-RMB-drag, Shift-Ctrl-RMB

在节点之间建立链接，而不需要精确地选择插座。只需按住 Ctrl-RMB 从一个节点拖动到另一个节点即可。使用的是离鼠标最近的节点，所以你甚至不需要精确地点击一个节点。

另外，使用 Shift-Ctrl-RMB 来显示输入和输出的菜单，这样可以更准确地进行连接。这在处理大节点树时非常有用，因为你可以很容易地进行连接，而不需要放大和缩小。

懒人混合(Lazy Mix)¶

参考

快捷键(Hotkey)

Alt-RMB -drag

在按住 Alt-RMB 的同时，通过从一个节点拖动到另一个节点，合并两个节点的输出。一个Mix节点或Mix Shader节点将被添加，并且节点的输出将被连接到它。

合并(Merge)¶

快速添加Mix、Math、Z-Combine、Alpha Over、Mix Shader或Add Shader节点，这些节点将合并所选节点的输出。

与自动类型检测合并¶

参考

快捷键(Hotkey)

Ctrl-0 , Shift-Ctrl-= , Ctrl-Minus , Shift-Ctrl-8 , Ctrl-8 , Ctrl-斜杠 , Shift-Ctrl-逗号 , Shift-Ctrl-句号

所选节点的输出将根据所选节点的类型，使用Mix、Math、Mix Shader或Add Shader节点进行合并。

添加 Shift-Ctrl-=

将混合模式或数学操作设置为添加。选择了着色器后，将使用 Add 着色器。

乘 Shift-Ctrl-8

乘法混合模式或数学运算。

减去 Ctrl-Minus

减去混合模式或数学操作。

除以 Ctrl-斜杠

除法混合模式或数学运算。

混合 Ctrl-0

混合混合模式或选择着色器时，将使用混合着色器节点。

大于 Shift-Ctrl-逗号

大于数学运算。

小于 Shift-Ctrl-句号

少于数学操作。

使用混合节点进行合并¶

参考

快捷键(Hotkey)

Shift-Ctrl-Alt-= , Ctrl-Alt-减号 , Shift-Ctrl-Alt-8 , Ctrl-Alt-斜杠

使用 Ctrl-Alt 和正确的键，无论选择什么类型的节点，都可以强制使用Mix节点进行合并。

使用Z-Combine节点合并¶

参考

快捷键(Hotkey)

Ctrl-小键盘句号

Z-Combine节点将被用于合并。如果可能的话 -- -- 图像和Z-深度输出将被连接起来。

使用Alpha Over Node合并¶

参考

快捷键(Hotkey)

Ctrl-Alt-0

Alpha Over节点将被用于合并。

使用数学节点进行合并¶

参考

快捷键(Hotkey)

Shift-Ctrl-= , Shift-Ctrl-Minus , Shift-Ctrl-8 , Shift-Ctrl-斜杠 , Shift-Ctrl-逗号 , Shift-Ctrl-句号

使用 Shift-Ctrl键 ，无论选择什么类型的节点，都可以强制使用Math节点进行合并。

批量改变混合模式/数学操作¶

参考

快捷键(Hotkey)

Alt-上 , Alt-下 , Alt-0 , Shift-Alt-= , Alt-Minus , Shift-Alt-8 , Alt-斜杠 , Shift-Alt-逗号 , Shift-Alt-句号

选择混合节点或数学节点，同时改变其混合模式或数学操作。使用上面指定的键。它们将相应地改变混合模式或操作。混合、加、减、除、小于、大于。使用 Alt-上 或 Alt-下 来查看所有可用的混合模式或数学操作。

变化混合系数¶

参考

快捷键(Hotkey)

Alt-Left, Shift-Alt-Left, Alt-Right, Shift-Alt-Right

选择的混合节点或混合着色器节点的变化系数。

  • 使用 Alt-右 将其增加0.1。

  • 使用 Alt-左 将其减少0.1。

  • 使用 Shift-Alt-右 将其增加0.01。

  • 使用 Shift-Alt-左 将其降低0.01。

  • 使用 Shift-Ctrl-Alt-左 将系数设置为0.0。附加快捷键是 Shift-Ctrl-Alt-0 。

  • 使用 Shift-Ctrl-Alt-右 将系数设置为1.0。附加的快捷键是 Shift-Ctrl-Alt-1 。

删除未使用的节点¶

参考

快捷键(Hotkey)

Alt-X

清理你的节点树。删除所有对最终结果没有贡献的节点。

交换链接(Swap Links)¶

参考

快捷键(Hotkey)

Alt-S

选择两个输出有链接的节点， Alt-S 将交换输出。原来链接到第一个节点的输出的东西现在会链接到第二个节点的输出，反之亦然。

Or, select one node with a single linked input -- Alt-S will cycle the link through the available input sockets. Or, select one node with two linked inputs -- Alt-S will swap the two links. If more than two inputs are linked, the two with matching types will be swapped (such as a Mix node's two color inputs).

重置背景墙(Reset Backdrop)¶

参考

快捷键(Hotkey)

Z

重新设置背景墙的位置和比例。

添加UV布局属性节点¶

参考

菜单(Menu)

Header ‣ Add menu

添加属性节点，属性设置为UV Layout。

添加顶点颜色属性节点¶

参考

菜单(Menu)

Header ‣ Add menu

添加一个属性节点，属性设置为顶点颜色。

着色器查看器(Shader Viewer)¶

参考

快捷键(Hotkey)

Shift-Ctrl-LMB

着色器的查看节点。 Shift-Ctrl-LMB ，该节点的输出将被链接到新创建的Emission着色器和Material 输出。这是预览纹理的一个简单方法。

选定的框架(Frame Selected)¶

参考

快捷键(Hotkey)

Shift-P

选择节点并通过点击 Shift-P 将其 "包裹 "在Frame节点中。紧接着点击 F6 来设置Frame的颜色和标签。

重载图片(Reload Images)¶

参考

快捷键(Hotkey)

Alt-R

节点树中使用的所有图像（图像输入、纹理）都可以通过点击 Alt-R 重新加载。

切换节点类型¶

参考

快捷键(Hotkey)

Shift-S

将所选节点的类型改成其他类型。点击 Shift-S ，你会得到与 Add ‣ Node 菜单完全相同的菜单排序。选择新的类型。

复制设置(Copy Settings)¶

参考

快捷键(Hotkey)

Shift-C

将活动节点的设置复制到所有选中的同类型节点。

复制标签(Copy Label)¶

参考

快捷键(Hotkey)

Shift-C

Copy labels all selected nodes based on various criteria. Labels can be copied from labels (names) of active node Shift-V, or from names (labels) of nodes that are linked to selected ones or from the names of sockets that the selected nodes are linked to. All options will be revealed in submenu after hitting Shift-C.

清除标签(Clear Label)¶

参考

快捷键(Hotkey)

Alt-L

清除所选节点的标签。

修改标签(Modify Label)¶

参考

快捷键(Hotkey)

Shift-Alt-L

批量更改所选节点的标签。添加文字到开头，到结尾，替换部分文字。

添加纹理设置¶

参考

快捷键(Hotkey)

Ctrl-T

选择任何一个着色器节点， Ctrl-T ，就会添加一个带有控制坐标的图像纹理节点。如果你选择了任何一个纹理节点，只添加纹理坐标和贴图节点。背景着色器将获得一个环境纹理节点，并生成映射。

在输出中添加重路由¶

参考

快捷键(Hotkey)

斜杠

重路由节点将被添加并链接到每个选定节点的每个输出。

活动链接到选定的¶

参考

快捷键(Hotkey)

Backslash , K , Shift-K , ' , Shift-' , Shift-; , Shift-;

根据各种标准将活动节点与选定的节点连接起来。

  • 反斜线 -- -- 调用主 链接活动到所选 菜单。

  • K ----所有选中的链接都是活动的。使用 Shift-K 强制替换现有链接。

  • Shift-' -- 仅链接到与活动节点名称/标签相同的选定节点（ Shift-' 替换现有链接）。

  • ; -- 当输出的名称与所选节点的名称或标签匹配时，选择链接。用于替换源的时候很方便。例如，将图层渲染成图像（多层EXR）。

节点对齐(Align Nodes)¶

参考

快捷键(Hotkey)

Shift-=

节点的水平或垂直对齐。与 S X 0 或 S Y 0 相同，但节点之间的间距要均匀。

框架内选择（父母/子女）¶

  • ] -- 选择被选中的Frame节点中的所有节点。

  • [ -- 选择所选节点被包裹的框架节点。

分离输出(Detach Outputs)¶

参考

快捷键(Hotkey)

Shift-Alt-D

选定节点的输出离开链接的输入。

链接到输出节点¶

参考

快捷键(Hotkey)

O

在合成中--链接到合成输出。在材料中 -- -- 链接到材料输出节点。

添加图像序列¶

参考

菜单(Menu)

Add ‣ Input 菜单用于合成节点，或 Add ‣ Texture 菜单用于着色器节点

只需在文件浏览器中选择一个序列中的图像，它将自动检测序列的长度，并适当设置节点。

添加多个图像¶

参考

菜单(Menu)

Add ‣ Input 菜单用于合成节点，或 Add ‣ Texture 菜单用于着色器节点

简单地允许你选择一个以上的图像，并为每个图像添加一个节点。(对于导入多张渲染图或叠加图像的渲染图很有用。)
