场景属性¶

场景¶

参考

面板

Properties ‣ Scene ‣ Scene

相机

Used to select which camera is used as the active camera. You can also set the active camera in the 3D Viewport with Ctrl-Numpad0.

背景场景

可使用一个场景作为背景，当想专注于动画的前景，且不想背景元素碍事时这很有用。

该场景可以有自己的动画、物理模拟等，但如果用户要编辑其中的任何内容，必须从 场景 数据块菜单中选择它。

可以使用一个布景作为另一个场景的背景(可以嵌套)。所以用户可以使用现有场景作为新场景的背景，并在其基础上再做补充。

Tip

当blend文件包含这个可以在很多地方重复使用的环境时，还可以结合 关联到场景 一起使用。

活动剪辑

活动影片剪辑用于约束以及视图显示。

单位¶

参考

面板

Properties ‣ Scene ‣ Units

单位系统
无

使用与现实世界无关的单位，实际上这与 公制 相同，只是没有单位名称。

公制

在场景中使用公制单位系统。

英制

在场景中使用英制单位系统。

缩放单位

在内部单位和用户界面中显示的值之间进行转换时使用的比例系数。 当在微观或天文尺度上建模时，可以改变该系数。

Note

这仅影响用户界面中显示的值，而不影响内部的行为方式。 例如，物理模拟不考虑单位比例。

分离单位

使用 公制 或 英制 时，显示多个单位。如 2.285m 会显示为 2m 28.5cm 。

旋转
角度

在用户界面以角度为角的计量单位。

弧度

在用户界面以弧度为角的计量单位。

长度
自适应

用于特定值的单位取决于值的大小。 例如，某些值可能显示为 23cm ，而其他值则显示为 10km 。

米/厘米/英尺/...

一个固定的单元，用于用户界面中的所有长度。

质量

见 长度 。

时间

见 长度 。

音频¶

参考

面板

Properties ‣ Scene ‣ Audio

Options to control global audio settings. To control how sounds is played back from within Blender, see the audio settings in the Preferences.

音量

该场景的音量。

间距模型

根据距离更改声音衰减的计算方式。 物理上最正确的是 反向 模型，但也可以选择线性和指数衰减。 钳制模式将音量限制为低于100%(1.0)，这意味着如果距离小于参考距离，则音量始终为100%。 有关每个选项的确切描述，参见 OpenAL文档 。

多普勒速度

多普勒效应计算的声音传播速度。典型数值为空气中343.3 m/s，水中大约1560 m/s.

多普勒系数

控制多普勒效应作用强度。你可以放大或缩小音调的变化，但物理真实系数为1.0.

更新动画缓存

更新音频动画缓存。这在你意识到音频存在瑕疵时很有用。
