集合¶

参考

模式

物体模式

面板

Properties ‣ Object Properties ‣ Instancing ‣ Collection

实例化集合 用于为每个其它物体的实例创建实例化集合。集合可能包含动画、带物理模拟的物体和其它嵌套的集合。

基础用法¶

创建一个集合：

  • 创建一个新集合(可在大纲视图中完成)

  • 关联需要被实例化的物体为新创建的集合的一部分。

创建一个新集合实例：

  • 添加(Add) ‣ 集合实例(Collection Instance)

此时，将出现集合的实例和 空物体。 您可以复制空物体，并为每个空物体保留 实例化集合 设置。 这样，您可以非常轻松地获得关联数据的多个副本。

集合和动态关联¶

参见 追加与关联 以了解如何将来自另一个blend文件的数据动态关联到当前文件中。 可以将集合从一个blend文件动态关联到另一个blend文件。 执行此操作时，在创建控制集合实例出现位置的物体之前，关联集合不会出现在场景中的任何位置。

创建一个真实的实例集合¶

If you want to make further edits on an instanced collection select the Instance Collection. Then call Make Instances Real to convert the collection into regular objects that can be transformed and animated normally.

Note

如果实例化集合由外界文件关联而来，物体数据(网格，材质，纹理，变换)将始终从源合集关联而来。但各类物体的父子关系不复存在。
