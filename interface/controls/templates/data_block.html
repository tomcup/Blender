数据块菜单¶

一组用于关联 数据块 的菜单按钮。编辑被关联的数据块之后，会同步更新到所有 数据用户 。

带有搜索框的数据块菜单。¶

类型

Shows an icon indicating the data-block type. It opens up the following pop-up menu. The data-block can be dragged from here e.g. to drag a material onto an object in the 3D Viewport or into a 数据ID field.

列表

当前文件或关联进来的可用数据块列表。菜单选项左侧会显示预览图标，下侧的搜索框用于根据名称搜索选项。

名称

显示关联的数据块的内部名称，同时可作为常规文本框进行编辑。如果该名称已存在，Blender会自动添加后缀如 ".001" 。

User Count

显示该数据的 数据用户 数量。单击使之独立成为单用户副本，仅与当前活动物体/物体数据关联。

伪用户(盾牌图标)

即使该数据块没有 真实用户 ，当 “F” 显示在名字列表旁边时，也会随blend文件一起保存。

保持本地(链接图标)

Todo <2.79.

新建/添加(文件图标)

新建数据块，或者复制当前数据块并应用。

Open File (folder icon)

打开 文件浏览器 。

Unpack File (bin icon)

将当前文件打包的文件 解包 为外部文件。

Unlink Data-block X

清除关联， Shift-LMB 重置用户数为0，将数据从文件中彻底删除。

有时需用到一 列 数据块 (如对一个物体使用多个材质)。

See also

数据系统章节 有关于数据块更深入的介绍。

预览¶

带有预览的数据块菜单。¶

位于工具栏，预览图更大的数据块菜单。(如绘制用笔刷选择菜单)

数据ID¶

数据ID框。¶

数据ID是左侧带有图标的文本框，单击显示灰色下拉菜单。数据ID是物体独一无二的名称，用于引用数据块，也因此Blender不允许两个同一类型的物体有相同ID(相同名称)。如果数据ID已被使用，Blender会自动追加名称后缀以避免冲突(比如 "Cube.001")。

显示数据ID的菜单包含以下元素:

类型

左侧的图标指示接受的数据了类型。

名称

文本框带匹配查找功能。使用 Tab 自动补全为匹配的名称。如果存在多个匹配项，需继续输入。如果用户输入非法名称，则该值不会变化。

列表

用于直接选择数据块。

吸管

一些数据ID框右侧带有 吸管工具 吸管图标可用 。

删除 X

单击右侧的 X 按钮删除引用对象。

子ID¶

下拉菜单会列出相关类型的ID，用以根据物体类型选择属性或子物体。

子ID示例。¶

顶点组

如果 名称 文本框内物体为网格或晶格，会显示一个文本框用于选择顶点组。

骨骼

如果 名称 文本框内物体为骨架，会显示一个文本框用于输入 骨骼 名称，指定单个骨骼。

头/尾

选定骨骼后，会显示数值输入框，用于选择指向头端或尾端。滑块用于指定该点沿骨骼轴向插值位置。0表示指向骨骼头/根部，1表示指向尾/尖端。

使用软骨骼形状

如该骨骼为 柔性骨骼，单击此按钮，使该点使用头尾之间B样条曲线曲率。
