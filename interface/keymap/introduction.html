通用快捷键¶

约定¶

键盘¶

在本手册呈现的字符快捷键，如下所示：

G

等同于小写的“g"。

Shift, Ctrl, Alt

特指修饰键。

Ctrl-W, Shift-Alt-A, ...

表示应同时按下这些键。

Numpad0 至 Numpad9, NumpadPlus

表示小键盘上的独立键位。

其他键位如它们的名字一样 Esc, Tab, F1 至 F12. 需要特别注意的是， Left, Right 等。

鼠标¶

本手册鼠标术语预定如下：

LMB

鼠标左键

RMB

鼠标右键

MMB

鼠标中键

Wheel

鼠标滚轮。

鼠标¶

Blender内置了两个选择物体的模式，左键选择和右键选择。

在过去，Blender使用右键单击选择来更清楚地区分选择和操作。在此模式下，RMB (鼠标右键)通常用于选择， LMB (鼠标左键)启动或确认操作。

如今，Blender用户可以继续选择使用以前的右键选择方法，或者使用与其他软件保持一致习惯的左键选择模式。

视频: 了解右键选择的好处。

悬停¶

当悬浮(当鼠标放在一个输入框上时)。

属性¶

  • Ctrl-C -- 复制输入框里的值(单个)。

  • Ctrl-V -- 粘贴值进输入框(单个)。

  • Ctrl-Alt-C -- 复制输入框里整个向量值或色彩值。

  • Ctrl-Alt-V -- 粘贴整个向量值或色彩值进输入框里。

  • RMB -- 打开上下文菜单。

  • Backspace -- 清除值(归零或清楚内容)

  • Minus -- 否定数值(乘以-1.0)。

  • Ctrl-Wheel -- 递增变化。

    对于弹出选项菜单按钮，将循环选择。

  • Return -- 选择菜单项或切换值。

  • Alt -- 在编辑时按住，会把操作应用到所有选择项中。(如物体，骨骼，视频片段)。

    这对于数值栏或开关操作非常有用。(译者注：如同时更改灯光亮度时，你可以同时选择几个灯光，然后按住alt键，同时调节灯光亮度)

动画¶

  • I -- 插入关键帧。

  • Alt-I -- 清除关键帧。

  • Shift-Alt-I -- 清除所有关键帧(移除所有F-曲线)。

  • Ctrl-D -- 添加驱动。

  • Ctrl-Alt-D -- 清除驱动。

  • K -- 添加插帧集。

  • Alt-K -- 清除插帧集。

Python脚本¶

  • Ctrl-C -- 在任何 操作按钮 上，复制该按钮的Python 命令进剪切板。

    这在Python 控制台或者在文本编辑器中写脚本时非常有用。

  • Shift-Ctrl-C -- 在一个属性栏上复制该操作属性的调用路径。(也适用于上下文菜单)。

    对写驱动脚本时非常有用。

  • Shift-Ctrl-Alt-C -- 在一个属性栏中复制 全部 数据块或属性值。

    需要注意在大多数情况下，最好根据正文而不是名称来访问值。

拖拽¶

  • Ctrl -- 拖动时吸附倍数步长。

  • Shift -- 精确控制拖拽的值。

  • Shift-Ctrl -- 伴随高精度的移动物体时的精确捕捉约束。

文本编辑¶

  • Home -- Go to the start of the line.

  • End -- Go to the end of the line.

  • Left, Right -- 把光标移动一个字符距离。

  • Ctrl-Left, Ctrl-Right -- 把光标移动一个单词距离。

  • Backspace, Delete -- 删除字符。

  • Ctrl-Backspace, Ctrl-Delete -- Delete words.

  • Shift -- Select while holding the key and moving the cursor.

  • Ctrl-A -- 选择所有文本。

  • Ctrl-C -- 复制选择的文本。

  • Ctrl-X -- 剪切选择的文本。

  • Ctrl-V -- 在光标的位置粘贴文本。

Confirm & Cancel¶

  • Esc, RMB -- Cancel.

  • Return, LMB -- Confirm.
