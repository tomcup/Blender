2D稳定面板¶

这项功能的目的是为了消除现有的现实世界中的镜头上的抖动处理。要激活二维稳定器，需要在面板中设置切换，另外还需要在剪辑显示弹出窗口中启用*显示稳定*。然后你需要设置一些跟踪点来检测图像的移动。

二维稳定板被用来定义用于拍摄的二维稳定的数据。此面板中有几个选项可用：您可以添加一个轨迹列表以确定横向图像移动，以及另一个轨迹列表以确定倾斜和缩放移动。根据这些轨迹的平均贡献，计算补偿运动并应用于每个帧。

当镜头包括平移和移动时，稳定器趋向于将图像推到可见区域之外。这可以通过对有意的“预期”摄像机运动的动画参数进行补偿。

Note

要 激活 2D稳定器，你需要在面板中设置切换，另外你需要在 显示 面板中启用 显示稳定器。

选项¶

2D稳定面板。¶

锁定在框边

锚稳定参考点：其他帧将根据该帧的位置、方向和比例进行调整。选择一个帧编号以最佳方式展示你的主题。

稳定型
旋转

除位置外，稳定检测到的围绕 旋转枢轴点 的旋转，这是所有位置跟踪点的加权平均值。

缩放

补偿相对于旋转中心的任何缩放的变化。

Tracks for Stabilization
位置

轨道的列表以用于补偿相机的跳跃，或位置移动。

旋转/缩放

轨道的列表以用于补偿相机倾斜和比例的改变。

自动缩放

查找，当应用到镜头，将消除图像边界附近的所有空的黑色边框最小的比例参数。

最大

限制自动缩放的量。

预估位置X / Y

已知的相对原始的拍摄偏移，将被减去，例如用于摇摄。

预期旋转

存在于原来拍摄的图像旋转，将得到补偿，例如用于故意倾斜。

预期放大

显式缩放结果帧以补偿原始镜头的变焦。

影响

可以控制应用于视频的转换量。在某些情况下，不需要完全补偿相机跳跃。可以控制应用于胶片的稳定量。在某些情况下，您可能不想完全补偿相机的某些跳跃。请注意，这些“影响*”参数只控制稳定器计算的*补偿运动*，而不是通过“预期*”参数添加的有意运动。

插值

稳定器以子像素精度计算补偿运动。因此，最终的图像像素需要从几个相邻的源素材像素中导出。不幸的是，任何插值会导致一些轻微的软化和图像质量损失。

最新

No interpolation, uses nearest neighboring pixel. This setting basically retains the original image's sharpness. The downside is we also retain residual movement below the size of one pixel, and compensation movements are done in 1 pixel steps, which might be noticeable as irregular jumps.

双线性

相邻像素之间的简单的线性内插。

双立方

最优质的插值，最昂贵的计算。
