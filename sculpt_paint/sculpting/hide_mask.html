隐藏 & 遮罩¶

在雕刻的过程中，模型的部分区域可能隐藏在了一些网格的后面或者太靠近其他部分。要在该种情况继续工作，隔离部分网格以进行雕刻是非常有用的。这可以通过隐藏部分网格或者对不需要进行雕刻的区域进行遮罩来实现。

隐藏¶

参考

模式

雕刻模式

菜单

雕刻

可以在雕刻模式中隐藏网格的某些部分，以提高视窗性能并访问原本因为被其他部分遮挡难以访问的部分网格。

隐藏的面不能雕刻。在编辑模式和雕刻模式之间隐藏的状态是共享的(即，在一种模式下隐藏/取消隐藏也会影响另一种模式)。

Show All 显示所有 Alt-H

显示所有隐藏的部分。

显示边界盒 Shift-H

显示选择框内的网格的隐藏部分。

隐藏框选 H

To hide a part of a mesh inside the selection. This works similar to the Box Select tool.

隐藏遮罩

隐藏所有被遮罩的顶点。

遮罩¶

遮罩用于控制网格的哪些区域会受到雕刻的影响。

被遮罩的部分显示为黑色。¶

笔刷¶

To edit the mask, select the Mask Brush from the Brush panel.

编辑¶

参考

模式

雕刻模式

菜单

Mask

可以在整个模型中编辑遮罩。使用 A 打开饼菜单以选择不同的操作。

反转遮罩 Ctrl-I

反转现有的遮罩。

填充遮罩

使用为1的数值来填充整个遮罩。

清除遮罩 Alt-M

Fills the mask with a value of 0. To completely remove the mask data see Clear Sculpt-Mask Data.

框选遮罩 Alt-M

其工作方式类似 框选 工具，它可以创建一个矩形的遮罩区域。按住 Shift 来清除选择区域内的遮罩。

套索遮罩 Shift-Ctrl-LMB

可用于创建自由形式的遮罩，与 套索 选择类似。

Tip

要使用 套索遮罩 工具清除区域的遮罩，首先反转遮罩，应用 套索遮罩，然后再将遮罩反转回来。

遮罩滤镜

遮罩滤镜是应用于整个绘制遮罩的操作。

类型
平滑/锐化遮罩

Changes the crispness of the mask edge.

生长/收缩遮罩

Changes the size of the mask.

提高/降低对比度

Changes the contrast of the mask.

迭代

The number of times that the filter is going to be applied.

自动迭代次数

根据雕刻的顶点数使用自动迭代次数。

按拓扑展开遮罩 Shift-A

以均匀的方式从活动顶点向外辐射遮罩。

按曲率展开遮罩 Shift-Alt-A

创建从活动顶点向外辐射的遮罩，同时遵循网格的曲率。

遮罩提取

从被遮蔽的几何体创建副本网格物体。

阈值

最小遮罩值，用以考虑从原始网格中提取面的顶点有效。

添加边界循环

Creates and extra boundary loop on the edges of the geometry, making it ready for adding a Subdivision Surface modifier later.

平滑迭代

应用于提取网格的平滑迭代次数。

投影至雕刻

将提取的网格投影到原始雕刻物体中。

提取为实心

向新创建的网格物体添加 实体化修改器。

遮罩切片

从网格中删除遮罩顶点。

阈值

最小遮罩值，以考虑从原始网格中提取面的顶点有效

填充洞面

使用可能由 遮罩切片 操作产生的几何体填充凹孔。

切片到新物体

从被遮蔽的几何体创建新的物体。

污迹遮罩

基于几何型腔和尖锐度生成遮罩。

Display Settings¶

参考

模式

雕刻模式

叠加窗口

视图覆盖 ‣ 雕刻 ‣ 遮罩

The mask display can be toggled as a viewport overlay. In the overlay popover, the opacity of the mask overlay can be adjusted to make it more or less visible on the mesh.

Clear Sculpt-Mask Data¶

参考

模式

物体/编辑模式

菜单

属性 ‣物体数据 ‣ 几何数据 ‣ 清除雕刻遮罩数据

完全清除网格的遮罩数据，虽然不会有很大的好处，但是如果不再使用遮罩，这么做可以提高雕刻性能。
