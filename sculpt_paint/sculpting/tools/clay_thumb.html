Clay Thumb¶

Reference

Mode

Sculpt Mode

Tool

Toolbar ‣ Clay Thumb

This brush simulates deforming clay with the finger, accumulating material during the stroke. It has a plane that tilts during the stroke in the front part of the brush to achieve this effect.
