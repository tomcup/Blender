默认配置¶

首次启动Blender时，可以在 启动画面 的互动区域初始化偏好设置来配置Blender中的交互方式。

Note

这些设置也可以之后在 用户设置 进行修改。

语言

用于翻译用户界面的语言。该列表根据翻译的完成度分为几类。更多语言相关设置见 翻译设置。

快捷键

默认Blender 键位映射 。请注意,本手册假定您使用"Blender"键位映射。

Blender

默认的键位映射。点击 这里 了解更多。

Blender 2.7x

这种键位映射是为了兼容近期的Blender重要版本，适用于更新版本后不愿意使用最新版键位映射的人群。

行业兼容

这种键位映射兼容常见的同类软件，适用于使用不同软件的人群。点击 这里 了解更多.

鼠标键选择

配置通过鼠标左键或鼠标右键在Blender中选择物体。

空格键

配置空格键 Spacebar 。这些和其他快捷键可以通过 键位映射设置 修改。

播放

开始播放 时间线 。这个选项适用于动画或视频剪辑。

工具

在鼠标箭头下方打开工具栏来快速切换工具。这个选项适用于建模或套索。

查找

Opens up the Menu Search. This option is good for someone who is new to Blender and is unfamiliar with its menus and shortcuts.

主题

选择Blender的白色或黑色主题。可以在 主题 里配置个性化主题。

Load Previous Version Settings

Copies preferences and startup files from the previous version of Blender and then loads them.

The settings need to be imported from previous versions because the configuration files of each Blender version are stored in separate folders. Refer to the Blender目录布局 page for the location of these folders.

Blender的默认配置包括两部分：

偏好设置

偏好设置 文件保存了键位映射，插件主题和其他选项。

启动文件

启动文件 保存了场景，工作区，启动以及加载新文件(文件 —> 新建)的界面。

保存默认设置¶

用户设置会自动保存。

通过 文件(File) ‣ 默认(Defaults) ‣ 保存启动文件(Save Startup File) 可以修改默认启动文件。参见 启动文件。

加载初始设置¶

用户可以恢复自定义设置为Blender默认设置：

偏好设置

偏好设置 可以恢复初始设置。

启动文件和偏好设置

文件(File) ‣ 默认(Defaults) ‣ 加载初始设置(Load Factory Settings)。

Note

恢复初始设置后，设置不会自动保存。

详情见 管理偏好设置 。
