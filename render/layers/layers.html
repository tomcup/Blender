视图层¶

渲染可以被分成不同的层，然后将它们重新组合在一起。

一些示例用法是将合成效果分别应用于角色，或分别为模糊背景和前景层用于景深，或渲染同一场景的不同光照变化。

使用视图层还可以避免每次更改时重新渲染整个图像，从而允许您仅重新渲染所需的图层。

视图层¶

视图层。¶

在屏幕顶部，有一个激活的场景中所有视图层的列表。

Name

The name of the active view layer, click to edit the name.

添加视图层

将向激活的场景添加新视图层。

New

Adds a new view layer.

Copy Settings

Adds a new view layer with all the settings of current view layer.

Blank

Adds a new view layer with all collections disabled.

删除视图层

将从激活的场景中删除选择的视图层。

Note

A scene must have at least one view layer.

视图层面板(这里显示的是Eevee渲染引擎)¶

视图层面板显示活动的视图层的设置。

用于渲染

渲染期间将使用活动的视图层。

渲染单个层

仅渲染激活的视图层。

Note

从命令行渲染时，将忽略此选项。

See also

此面板中显示的其他选项对于每个渲染引擎都是不同的。有关每个渲染引擎的选项，请参见 通道 。

用法¶

每个视图层都有一组相关的 集合 。只要该集合也可见，则该视图层中将显示其中一个关联集合上的对象。

Warning

仅渲染可见场景中的对象。因此，如果只有集合1可见并且您的渲染层集指定仅渲染集合2和3，则不会渲染任何内容。

集合¶

每个集合都可以调整渲染引擎渲染内部对象的方式。基于渲染引擎，可以设置不同的选项。

集合/视图层设置¶

从视图层禁用

从活动视图层中排除该集合。仅在该集合中的对象不会为活动视图层进行渲染。

在视图层中启用

将该集合加入活动视图层。集合中的对象将使用活动视图层进行渲染。

设为仅间接作用

改集合中的对象仅通过阴影和反射间接地对最终图像做出贡献。

清除仅间接作用

Clear the Set Indirect Only flag. Objects inside this collection will contribute normally to the final image.

设为阻隔

该集合中的对象将在活动视图层中生成一个阻隔/遮罩。

清除阻隔

清除“设为阻隔”标志。

视图层¶

排除

场景图层在所有渲染层之间是通用的; 但是，有时省略特定渲染层的某些对象是有用的。

滤镜(过滤)¶

Include
Environment

启用在最终渲染中渲染 环境 。

Ambient Occlusion

启用在最终渲染中渲染 环境光遮蔽(AO) 。

Surfaces

启用在最终渲染中渲染实体材质。

Hair

启用在最终渲染中渲染头毛发。

Volume

Disables rendering Volumes in the final render.

覆盖¶

材质覆盖

覆盖该视图层中的所有材质。

采样

视图层采样次数覆盖场景采样次数。由采样面板中的 图层采样 控制。
