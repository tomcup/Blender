骨骼组¶

参考

模式

姿态模式

面板

Properties ‣ Armature ‣ Bone Groups

菜单

姿态 ‣ 骨骼组 ‣ ...

该面板用于创建、删除和编辑骨骼组。骨骼组可用于选择或将颜色主题分配给一组骨骼。例如，将绑定的左边设为蓝色，绑定右边部分为红色。

激活的骨骼组

骨骼组的( 列表视图 )。

色彩设置¶

你可以为组指定“颜色主题”(每个骨骼都会有这些颜色)。记住你必须启用 颜色 复选框(在 视图显示 面板)才能看到这些颜色。

Bone Color Set 骨骼配色集

选择菜单。

  • 默认颜色：默认(灰色)颜色。

  • nn - 主题配色集(Theme Color Set): 其中包括二十个Blender预设的主题。

  • 自定义配色集：一组自定义的颜色，这是一组特定的颜色。

常规

第一个颜色按钮是未选择骨骼的颜色。

选择

第二个颜色按钮是所选骨骼的轮廓颜色。

活动项

第三个颜色按钮是激活骨骼的轮廓颜色。

一旦你改变了一种颜色，它就被切换到 自定义配色集(Custom Set) 选项。

指定和选择¶

在3D视图中，使用 姿态 ‣ 骨骼组 菜单项，和/或弹出菜单 骨骼组 ，快捷键 Ctrl-G。：

指定

将选定的骨骼指定给激活的骨骼组。重要的是要注意骨骼只能属于一个组。

移除

从激活的骨骼组中移除选定的骨骼。

选择

选择激活的骨骼组中的骨骼。

取消选择

取消选择激活骨骼组中的骨骼。

See also

可在 关系面板 将单一的骨骼指定给一个组。

See also

使用 Selection Sets 插件可以调用属于不同骨骼组的骨骼。
