用法¶

可以通过对应的上下文菜单，快捷键，复制粘贴或直接键入属性值将驱动器添加到属性中。

添加驱动器后，通常会在 驱动器编辑器 中，或者通过从属性上下文菜单调用的简化的 编辑驱动器 弹出窗口进行修改。

添加驱动器¶

参考

菜单

上下文菜单(右键菜单) ‣ 添加驱动器

快捷键

Ctrl-D

向属性添加驱动器的常用方法是 RMB 单击属性，然后在上下文菜单中选择 添加驱动器。还可以通过将鼠标置于属性上方并按下 Ctrl-D 来添加驱动器。

此操作会向驱动器添加单个变量(需要填写)，并弹出 编辑驱动器 窗口。

驱动器编辑器¶

参考

菜单

上下文菜单(右键菜单) ‣ 驱动器编辑器

显示一个弹出窗口，允许编辑驱动器的自定义表达式和输入变量，而无需打开完整的 驱动器编辑器 。

许多驱动器无需使用它们的 函数曲线 组件，这种简化的界面就足够了。

打开驱动编辑器¶

参考

菜单

上下文菜单(右键菜单) ‣ 打开驱动器编辑器

使用 打开驱动器编辑器 打开一个新窗口，然后选择与该属性关联的驱动器。

复制和粘贴¶

参考

菜单

上下文菜单(右键菜单) ‣ 复制驱动器

菜单

上下文菜单(右键菜单) ‣ 粘贴驱动器

可以通过上下文菜单(右键菜单)复制和粘贴驱动器。添加具有相同设置的驱动器时，节省修改设置的时间。

复制为新驱动器¶

参考

菜单

上下文菜单(右键菜单) ‣ 复制为新驱动器

通过使用输入属性的 复制为新驱动器 上下文菜单选项，然后通过 粘贴驱动器 将结果粘贴到输出属性上，可以快速创建将属性值设置为其他属性值的驱动器。

也可以使用编辑器面板中的 粘贴驱动器变量 按钮为现有驱动程序添加新的驱动器变量。

表达式¶

这是使用脚本表达式添加驱动器的快速方法。首先单击要添加驱动器的属性，然后键入 # 和脚本表达式。

一些例子：

  • #frame

  • #frame / 20.0

  • #sin(frame)

  • #cos(frame)

删除驱动器¶

参考

菜单

上下文菜单(右键菜单) ‣ 删除驱动器

菜单

上下文菜单(右键菜单) ‣ 删除单个驱动器

快捷键

Ctrl-Alt-D

删除与属性关联的驱动器，用于单个选定属性，或子通道，或向量的所有分量。
