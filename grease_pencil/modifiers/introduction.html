简介¶

参考

面板

属性 ‣ 修改器

蜡笔有自己的一组修改器。 修改器是以非破坏性方式影响物体的自动操作。 使用修改器，您可以自动执行许多手动操作太繁琐的效果，否则，并且不会影响物体的基本几何体。

它们的工作原理是更改物体的显示和渲染方式,但不能更改可以直接编辑的几何体。您可以向形成修改器堆栈的单个物体添加多个修改器,如果希望将其更改永久化,则可以 应用 修改器。

蜡笔修改器面板。¶

蜡笔有三种类型的修改器:

生成

生成 修改器组包括构造工具，可以更改物体的一般外观或自动将新几何图形添加到物体。

变形

变形 修改器组仅更改物体的形状而不添加新几何体,

颜色

颜色 修改器组更改物体颜色输出。

界面¶

面板布局(厚度修改器作为示例)。¶

如同网格的修改器，每个修改器的界面共享相同的基本组件。

查看 修改器界面 来获得更多的信息。

Note

蜡笔笔触与网格不同，仍然无法在该位置直接编辑。

过滤器影响¶

大多数蜡笔修改器共享一些特殊属性，这些属性仅限制对特定项目的影响。

图层

仅将效果限制为一个图层或共享相同传递索引的任何图层。

材质

将效果限制为仅对共用相同材质或通道编号的材质。

顶点组

仅将效果限制为顶点组。

Custom Curve

When enabled, use a custom curve to shape the effect along the strokes from start to end points.

反转切换 <-> 允许您反转过滤器行为。
