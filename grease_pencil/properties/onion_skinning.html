洋葱皮¶

洋葱皮在当前帧之前和之后显示关键帧的鬼影，允许动画制作者在动画序列中做出决定。

The main switch to show/hide Onion Skinning is in the Viewport Overlays, but Grease Pencil Onion Skinning is per-layer and the visibility can be toggle in the layer list. See 2D Layers for more information.

洋葱皮面板。¶

选项¶

模式
关键帧

显示由 前/后 设置确定的范围内的关键帧。

帧

显示由 前/后 设置确定的范围内的帧。

选择

仅显示在摄影表中手动选择的关键帧上。

不透明度

控制叠加帧的透明度。

Filter by Type

筛选要在洋葱皮范围内显示的帧类型。

关键帧之前/之后

设置要在当前帧之前和之后显示多少帧或关键帧，取决于 模式 。

自定义颜色¶

之前/之后

在鬼影帧上用于之前帧和之后帧的颜色。

显示¶

View in Render

在最终渲染图像中显示洋葱皮， 比如用于运动模糊效果。

渐隐

鬼影帧的不透明度越远离当前帧越来越淡。

循环

帮助处理循环动画，当您在动画的最后一帧时显示第一个关键帧/帧为叠加。
