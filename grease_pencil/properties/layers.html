2D图层¶

Reference

Mode

All Modes

Panel

Object Data tab ‣ Layers

Hotkey

Y

图层列表¶

蜡笔图层面板。¶

Grease Pencil objects each have a list of 2D layers for grouping and arranging strokes in a List view. Any stroke can only belong to a single 2D layer. There is always only one active layer in the list (the selected one). When you draw, the new strokes are added to the active layer. By default the view order of the layers in the viewport is top to bottom.

Every layer correspond to a channel in the Dope Sheet editor (in Grease Pencil mode). See Dope Sheet for more information. Layers can also be used together with Modifiers to only affects part of your drawing. See Modifiers for more information.

Tip

Sometimes the layers you are not working on can be a distraction. Activate Fade Layers in overlays to control the opacity of the non-active layers. See Overlays for more information.

图层名称旁边有四个图标按钮，用于控制图层的常见属性：

蒙版(蒙版图标)

Toggle the Masks visibility in the layer.

洋葱皮(洋葱皮图标)

切换使用图层 洋葱皮。

视图/渲染可见性(眼睛图标)

在视图和渲染中切换图层的可见性。

锁定(锁上图标)

切换图层不可编辑。

在图层列表下方还有其他常用设置：

Blend

要执行混合的操作的图层。见 颜色混合模式 。

不透明度

用于设置图层的不透明度。

Use Lights

When enabled, the layer is affected by lights.

特定¶

复制图层

制作所选图层的精确副本，附加数字以区分其名称。

显示全部

打开列表中每个图层的可见性。

隐藏其他

关闭列表中除活动层之外的每个层的可见性。

锁定全部

锁定列表中所有图层的编辑性。

全部解锁

解锁列表中所有图层的编辑性。

Autolock Inactive Layer

Locks automatically the edition of every layer in the list except the active one. This way you avoid to make unwanted changes in other layers without the need to lock them every time.

向下合并

将所选图层与下面的图层合并，新图层保留靠下图层的名称。

复制层到物体

Makes a copy of the layer and move it to the selected Grease Pencil object.

Lock & Visibility General Controls¶

锁定(锁上图标)

切换活动图层是否是唯一可以编辑的图层。

可见性(屏幕图标)

切换活动图层是否是唯一可以编辑且可见的图层。

Masks¶

In a List view of layers affected by a layer mask. See Masks for more information.

调整¶

图层调整面板。¶

染色颜色/系数

为图层中使用的任何材质颜色着色的颜色。 Factor 控制要应用的色调颜色的数量。

笔画的宽度

覆盖图层中笔画粗细的宽度值。

通道编号

层编号值可与某些修改器一起使用，以限制仅对某些区域的更改。

更多信息见 修改器。

查看图层

Defines the View Layer to use for the Grease Pencil layer. If empty, the layer will be included in all View Layers. This is useful to separate drawings parts for compositing.

禁止锁定材质编辑

避免在图层中编辑锁定的材质。 禁用后，即使它们在材质列表中被锁定，也可以编辑任何材质。

关系¶

从属/类型

选择父物体，然后键入以操作图层。 该图层将继承父级的变换，这在剪纸动画时非常有用。

显示¶

自定义通道颜色

设置要在通道区域中使用的颜色 摄影表。

仅在关键帧上显示

仅当视图在实际帧中具有关键帧时，才会使视图在视口中可见。 例如，当您使用 填充 工具进行笔画处理并且只想查看实际帧中的笔画以避免填充不需要的区域时，这会有所帮助。
