选择¶

本页面讨论在编辑模式下曲线物体的特有的选择工具。曲面编辑也可以使用 界面选择 中提及的通用选择工具。

编辑模式 中的曲面选择非常类似于 NURBS曲线选择。 基本工具与 网格 相同，因此可以 LMB 点击选择一个简单的控制点，使用 Shift-LMB 点击添加到当前选择、框选等。

选择菜单¶

选择 菜单(3D 视图标题栏中的)比曲线的更简单...

All these options have the same meaning and behavior as in Object Mode and mesh Edit Mode.

All A

全部选择。

无 Alt-A

全不选。

反转 Ctrl-I

选择所有未选中的几何图形，并取消选择当前选中的组件。

框选 B

交互式框选。

刷选 C

交互式刷选。

随机选择

随机选择控制点。

间隔式弃选

选择第N个控制点。

选择相连元素 Ctrl-L

选择与当前选择相连接的控制点。

选择相似 Shift-G

选择与当前选择具有类似属性的控制点。

选择控制点行

选择整 行 控制点。

扩展选择/缩减选择

根据物体的父子关系选择物体。

间隔式弃选¶

参考

模式

编辑模式

菜单

选择 ‣ 间隔式弃选

此工具使用类似棋盘格的交替选择/取消选择模式。这只适用于已经选择了多个控制点的情况。

它的工作原理是改变当前的选择，这样只有从活动的控制点开始，只有每一轮的第n个控制点才会保持选中的状态。

弃选项

每次模式重复中弃选的元素数量。

选中项

每次模式重复中选择的元素数量。

偏移量

从起始点的偏移。

选择控制点行¶

参考

模式

编辑模式

菜单

选择 ‣ 控制点行

快捷键

Shift-R

这个选项在网格的 边环状选择 里可用, 因为它选择一个整体的控制点 行 , 基于激活的项(最后选择的). 你第一次按下 Shift-R, V行通过激活的点(含)将被添加到 当前 选择中. 如果你再次使用这个快捷方式，你会在这个点的U和V行间切换, 从选择中移除 一切别的。

扩展选择/缩减选择¶

参考

模式

编辑模式

菜单

选择 ‣ 扩展/缩减

快捷键

Ctrl-NumpadPlus, Ctrl-NumpadMinus

基于当前选定的控制点扩展或缩减所选内容。

扩展选择

对于每个选定的控制点, 选择和它相关联的 全部 (即二三或四).。

缩减选择

对于每个选定的控制点, 如果和它相关联的 全部 点都被选中了, 那也选中它。对于所有其他被选的控制点，取消选择他们。

这意味着两点:

  1. 第一，当一个曲面上 全部 控制点都被选择了, 那么什么都不会发生的(对于 缩减，所有相连的点总是被选择的，当然，扩展 不添加任何点)。相反地，没有控制点被选时发生一样的事情。

  2. 其次，这些工具不会“跳出”曲面，(它们不会跳到同一个物体的另一个面上)。
