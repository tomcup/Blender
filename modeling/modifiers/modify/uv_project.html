UV 投射修改器¶

投射Blender的logo到猴头上。¶

UV投射 修改器的作用类似于幻灯片投影仪。它从控制器对象(例如 空物体 )的负Z轴发出UV映射，就像 "光线"照射到该对象一样，将其应用于该对象。

下载样例。

选项¶

UV 投射修改器。¶

UV 贴图

要修改的UV贴图。 默认为激活的渲染层。

宽高比 X/Y

改变图像的宽高比。只适用于当相机作为投影物体。

缩放 X/Y

缩放图像。只适用于当相机作为投影对象。

投影源

最多支持十个投射对象。 每个面将选择最接近且其表面法线对齐的投射物体。 投影从负Z轴发射(即照相机或灯直接向下)。 如果投影仪是相机，则投影将遵循其透视/正交设置。

投影物体

指定投影物体。

使用¶

常规项¶

UV投射 对使聚光灯更加多样化，创建贴花以打破重复非常有效。

通常，一个使用这个UV贴图的 图像纹理节点 将添加到修改器目标的材质。
