Transform¶

Correct Face Attributes

Adjust geometry attributes like UVs and Vertex Colors while transforming.

Keep Connected

Merge attributes connected to the same vertex while using Correct Face Attributes.

Tip

Keeping UVs connected is useful for organic modeling, but not for architectural modeling.

镜像¶

您可以使用 网格选项 中的 镜像 选项根据所选的轴线对称地变换顶点。当您变换一个元素(顶点、边缘或面)时，如果有精确的与轴线镜像的对应元素(在局部空间中)，它将根据所选轴线对称地进行相应的转换。

Note

镜像 的使用条件相当严格，所以很难使用。要获得(半)网格的准确镜像，镜像修改器 来得更加简单和方便。

拓扑镜像¶

Note

要使用 拓扑镜像 ，必须先至少启用三个 镜像轴向 中的一个。

在使用 镜像 选项时，网格几何体的顶点必须完美配对。如果其位置无法准确配对， 镜像轴 不会将其当做镜像顶点处理。

拓扑镜像 试图通过综合考虑顶点位置以及与几何体中其他顶点的相对关系，来确定哪些是镜像顶点。该工具通过审视整体拓扑结构，来确定是否将具体顶点认定为镜像点。这样的效果是，非对称顶点也可以作为镜像顶点，并当做镜像处理。

Note

几何体越复杂， 拓扑镜像 功能更加可靠。如果处理的是非常简单的几何体，如 立方体 或者 经纬球 ， 拓扑镜像 选项通常是不起作用的。

例子¶

举个例子，说说如何使用 拓扑镜像，打开一个新的Blender场景，然后删除3D窗口中默认的立方体，再添加一个猴头。

  1. 按下 Tab 键，切换猴头进入 编辑模式。

  2. 禁用 镜像轴向，稍稍移动猴头的几个顶点。

  3. 再启用 X向镜像，保持 拓扑镜像 取消勾选。

  4. 现在再次移动顶点， X向镜像 不起作用，被镜像的顶点不会随之修改。

  5. 如果接着勾选 拓扑镜像 ，再次移动相同顶点， X向镜像 会对随之操作镜像顶点，即使这些顶点不是完美对应的。

自动合并¶

参考

模式

编辑模式

菜单

侧栏 ‣ 工具 ‣ 选项 ‣ 自动合并

启用 自动合并 后，只要将一个顶点向另一个顶点靠近至小于设定 阈值 ，将自动合并这两个顶点。该选项仅影响交互操作（调整上一步操作 面板中的调整也被认定为交互操作）。如果顶点移向的位置存在多个顶点，只会与其中一个顶点执行合并操作。

拆分边和面

检测每个变换的边的交点，就地创建新的顶点并分割边与面(如果有的话)。

阈值

指定 自动合并 工具使用的最大顶点距离。
