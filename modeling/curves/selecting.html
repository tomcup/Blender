选择¶

本页面讨论在编辑模式下曲线对象的特定选择工具。曲线编辑还使用了通用的选择工具，如 界面选择 中所述。

编辑模式 下的曲线选择选项比网格少。主要是因为它只有一种可选元素类型，控制点(这里没有选择模式)。这些点比简单的顶点稍微复杂一些，但是，特别是贝塞尔曲线，一个顶点包含一个中心点和两个控制柄...

The basic tools are the same as with meshes, so you can select a simple control point with the LMB, add to current selection with Shift-LMB, Box Select B, and so on.

对于贝塞尔曲线需要注意一点：当选择主控制点时，两侧控制柄也会自动处于选定状态，将其作为一个整体拖动，不改变控制柄的角度。但是，当选中一个控制柄后，选中的只是这个点，这允许你修改控制向量...

请注意，与网格边不同，你无法直接选择一段曲线，而是只能通过选择组成该段曲线的所有控制点来实现。

选择菜单¶

对于曲线来说，所有的 "高级" 选择选项都位于3D视图标题栏的 选择 菜单里。

全部 A

全选。

无 Alt-A

无选择。

反转 Ctrl-I

选择所有未选中的几何图形，并取消选择当前选中的组件。

框选 B

互动框选择。

刷选 C

刷选择。

随机选择

随机选择控制点。

间隔式弃选

选择第n个控制点。

选择相连元素 Ctrl-L

选择与当前选择相连接的控制点。

选择相似 Shift-G

选择与当前选择具有类似属性的控制点。

选中/弃选首点/末点

切换第一个或最后一个控制点的选择。

选择下一项 /上一项

选择下一个或上一个控制点。

扩展选择/缩减选择

根据对象的父级-子级关系选择对象。

间隔式弃选¶

参考

模式

编辑模式

菜单

选择 ‣ 间隔式弃选

此工具应用交替选择/取消选择的检查器模式。这只适用于您已经选择了多个控制点的情况。

它的工作原理是改变当前的选择，这样只有从活动的控制点开始，只有每一轮的第n个控制点才会保持选中的状态。

弃选项

每次模式重复中弃选的元素数量。

选中项

每次模式重复中选择的元素数量。

偏移量

从起始点的偏移。

选择相连元素¶

参考

模式

编辑模式

菜单

选择 ‣ 选择相连元素

快捷键

L, Ctrl-L, Shift-L

L (或者 Ctrl-L 全选) 会扩选光标附近的控制点和相连的全部控制点，也就是，选中同属一条曲线的所有控制点。注意，对于贝塞尔曲线，选中一个控制柄后使用 L 会选中所有控制点以及与之相连的所有控制点。

选中/弃选首点/末点¶

参考

模式

编辑模式

菜单

选择 ‣ 选中/弃选首点 , 选择 ‣ 选中/弃选末点

这些操作用于切换曲线首末端点的选中状态。这可以用在快速寻找曲线的起始点(比如将曲线用作路径时)。

选择下一项 /上一项¶

参考

模式

编辑模式

菜单

选择 ‣ 选择下一项 , 选择 ‣ 选择上一项

该操作基于当前选中项，扩选下一个或上一个控制点(即沿曲线的前一个或后一个控制点)。对于闭合曲线，起点和终点不算作相邻顶点。

扩展选择/缩减选择¶

参考

模式

编辑模式

菜单

选择 ‣ 扩展/缩减选择

快捷键

Ctrl-NumpadPlus , Ctrl-NumpadMinus

它们的目的是根据目前选择的控制点来减少或扩大选中项。

扩展选择

对于每个选定的控制点，选择它的 所有 连接点(即一个或两个…)。

缩减选择

对于每个选定的控制点，如果 所有 与此点连接的点都被选中，则保持选中此点。否则取消选择。

这意味着两点:

  1. 当选择曲线的 所有 控制点时，什么都不会发生(对于 缩减选择 ，所有连接的点总是被选择，当然，此时 扩展选择 不能添加任何点)。相反,在没有选择控制点时也一样。

  2. 其次，这两个工具的作用范围不会 "跳出" 曲线( 决不会从物体的一条曲线 "跳到" 另一条曲线上)。

拾取最短路径¶

参考

模式

编辑模式

菜单

Menu Search ‣ Pick Shortest Path

快捷键

Ctrl-LMB

选择两个控制点之间的曲线段: 活动点和光标下的控制点。在闭合曲线的情况下，选择最短路径。
