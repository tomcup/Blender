映射值节点¶

映射值节点。¶

映射值用于缩放，偏移和限定值。

输入¶

值（明度）

标准化值输入。(引用在设置框中的矢量值)。

属性¶

偏移量

把偏移因数添加到输入值。

尺寸

对输入值进行缩放(相乘)。

使用最小值，最大值

启用此复选框将激活相应的功能。

最小值，最大值

Defines a range between minimum and maximum to Clamp the input value to.

输出¶

值（明度）

标准化值输出。

范例¶

Z深度通道¶

这在获取Z深度通道时尤其有用，映射值节点将Z通道值(依据场景不同，数值可能为20,30甚至500)匹配到(0,1)的范围之间，这使得在连接模糊节点时数值更符合模糊节点的有效值范围。

将值相乘¶

映射值节点也可对数值进行相乘操作以得到所需输出值。在如图所示中，时间节点输出30帧的时间范围均匀分布在0.0到1.0之间的数值。第一个 映射值节点将输入值乘以2，将输出值范围缩放到从0.0到2.0。 第二个 映射值节点从输入值减去1，得到一个从(-1.00到1.0)的数值范围，然后将其乘以150，将30帧时间范围数值映射到范围在(-150到150)的输出值区间。
