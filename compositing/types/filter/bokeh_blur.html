散景模糊节点¶

散景模糊节点。¶

散景模糊节点生成一个和散焦相似的散景类型模糊。不像散焦是在合成中对精确区域进行的处理，此节点可以通过应用 散景图 节点产生更多复杂的效果。

在支持OpenCL的情况下，计算区域的限定和遮罩可以进行一些执行优化。

输入¶

图像

标准化图像输入。

散景

只是供 散景图 节点的输入连接端口。

尺寸

模糊量的尺寸限制。尺寸可以由整个图像上用单一数值或可以控制的可变值来确定。为了以后便于使用，可变值必须被勾选。关于此节点如何使用的更多范例请参看如下部分。

边界框

此设置框在 方形遮罩 节点或 遮罩 输入节点去限定图像要应用模糊的区域。允许一小块图像区域被过滤可以在频繁调整画面效果时节省合成的时间，在开发节点系统时这将非常有用。

属性¶

变量大小

对输入图像使用可变模糊半径。

Max Blur

Max Blur is intended to act as an optimization tool by limiting the number of pixels across which the blur is calculated.

输出¶

图像

标准化图像输出。
