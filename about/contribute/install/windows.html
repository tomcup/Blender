Windows上安装¶

本指南包含以下主题：

  1. 安装Python (用于 转换 原始文件为HTML)

  2. 安装SVN与下载代码库

  3. 设置编译环境

安装Python¶

  1. 下载Windows下的 Python安装包 。 本指南使用的版本是3.6.x。

  2. 使用安装向导安装Python。记得勾选 "Add Python to PATH" 选项:

    该选项必须勾选，这样才能使用脚本编译手册。¶

    其余所有设置维持默认。

安装SVN与下载代码库¶

尽管还有很多Subversion客户端，在本指南中, 我们使用 TortoiseSVN。

  1. 下载Windows下的 TortoiseSVN 。

  2. 使用安装向导安装TortoiseSVN。在选择安装特性时，推荐启用 command line client tools ，这样你可以在命令行使用SVN(这没有任何害处，并且可能在你遇到麻烦的时候帮到你)。

  3. 一旦安装完成，创建一个新的文件夹，用于存储Blender手册所有相关内容。这里，我们使用 C:\blender_docs 。

  4. 打开新建的文件夹，右键单击，然后弹出菜单中选择 SVN Checkout... 。

  5. 在 URL of repository 文本框中，输入： https://svn.blender.org/svnroot/bf-manual/trunk/blender_docs 。

  6. 在 Checkout directory 文本框，输入： C:\blender_docs.

  7. 点击 OK ——几分钟过后代码库就下载下来了，具体耗时取决于你的网络连接。

设置编译环境¶

  • 打开命令行提示符。(使用管理员启动)

  • 进入刚刚SVN检出新建的 blender_docs 文件夹:

    cd C:\blender_docs

  • 夹中有一个名为 requirements.txt 的文件，包含了一系列需要安装的依赖关系。要安装这些依赖关系，可以使用 pip 命令:

    pip install -r requirements.txt

  • 如果一切顺利，完成后会显示以下信息：

    Successfully installed Jinja2 MarkupSafe Pygments Sphinx docutils sphinx-rtd-theme Cleaning up...

在安装过程中，可能会显示一些警告，但不要担心。但是如果发生任何错误，则可能会造成一些问题。

Note

时不时使用下面的命令确认你的依赖库是不是最新的:

pip install -r requirements.txt --upgrade

继续下一步: 编译。
