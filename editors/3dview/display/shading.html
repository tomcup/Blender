视图着色¶

可以调整3D视图的阴影以匹配手头的任务。 有几种模式可供选择。

Note

当场景的渲染引擎设置为工作台时，材质预览选项不可用。

线框¶

Shows the full scene by only displaying the edges of the objects (wireframes).

颜色
单个

使用单一颜色渲染整个场景。

物体

使用视图显示 物体 面板中的颜色，可为每个物体设置该颜色。

随机

将为场景中的每个对象选择随机颜色。

背景

背景如何在3D视图中显示。

主题

使用主题的背景颜色。

世界环境

使用世界视图显示选项。

视图

为3D视图的背景选择自定义颜色。

选项
透视

渲染场景至透明。 使用滑块可以控制场景的透明度。 在线框模式下，可以调整后面的线的不透明度。

轮廓

在视图中渲染物体的轮廓。 轮廓的颜色可以调整。

实体¶

在实体模式下显示场景。 此模式使用工作台引擎渲染3D视图。 光照 ， 颜色 和 选项 可以在工作台渲染引擎章节。

背景

背景在3D视图中的显示方式。

主题

使用主题的背景颜色。

世界环境

使用世界视图显示选项。

视图

为3D视图的背景选择自定义颜色。

材质预览¶

使用 Eevee 和 HDRI 环境渲染 3D 视口。此模式特别适用于预览材质和纹理绘制。用户可以选择不同的光照条件来测试材质。

光照
场景灯光

渲染场景时使用场景中的灯光。

场景世界

渲染场景时使用场景的世界。 当关闭时，将使用下一个选项构建一个世界。

HDRI环境

用于点亮场景的环境贴图。

旋转

环境在Z轴上的旋转。

强度

环境光照的强度。

World Opacity

The opacity level of the HDRI will be rendered as background in the 3D Viewport.

Blur

Factor to unfocus the HDRI Environment, note that this does not change the diffusion of the lighting, only the appearance of the colors.

渲染通道

显示另一个渲染通道，而非组合渲染。可用于分析和调试几何形状、材料和照明。

渲染¶

Render the 3D Viewport with the scene Render Engine, for interactive rendering. By default the scene lights are used for lighting. An HDRI environment can be used as well, with the same options as Material Preview mode.

渲染通道

显示另一个渲染通道，而非组合渲染。可用于分析和调试几何形状、材料和照明。
