文件浏览器¶

文件浏览器用于所有与文件相关的操作。这些包括:

  • 打开和保存 blend文件。

  • 浏览其他 blend文件时，当附加或链接数据块时，见 关联库.

  • 导入/导出其他文件格式。

  • 为现有的文件路径(图像，视频，字体...)选择新的位置。

使用此编辑器最常见的方法是通过模态操作(如打开或保存一个blend文件)。它将显示最大化，等待操作完成，然后关闭并返回到以前的屏幕布局。

Note

您始终可以在文件浏览器中选择多个条目，最后选择的条目将被视为活动条目。如果调用操作需要一个单独的路径(例如 打开 一个主blend文件)，它将获取活动项的路径，其他选定的路径将被忽略。

您还可以像任何其他编辑器类型一样保持文件浏览器打开，以便浏览文件系统。其主要目的是能够拖放媒体文件：

  • 将图像放入 3D视图 (设置为背景或应用为材质纹理)。

  • 媒体文件放入 视频编辑.

文件浏览器。¶

标题栏¶

导航

Icon buttons for navigation of files.

左箭头键 Backspace, Alt-Left

移动到上一个文件夹。

右箭头 Shift-Backspace, Alt-Right

移到到下一个文件夹。

向上箭头 P, Alt-Up

移动到父级目录。

循环箭头 R, NumpadPeriod

刷新当前文件夹。

创建新的文件夹 I

将要求您确认并在当前目录中创建一个新目录，在主视图中移动到它，并让您输入它的名称。

显示模式

控制文件的显示方式。

  • 垂直列表 以垂直列表显示文件和文件夹。

  • 水平列表 以水平列表显示文件和文件夹。

  • 缩略图 显示 预览.

排序

按四种建议方法之一(按字母顺序、按类型、按日期或按大小)对项目进行排序:

Show Hidden H

显示隐藏文件 (从 . 开始).

视图菜单¶

Display Size

缩略图的大小，或列的宽度。

递归

一次展开显示的目录级别数。

  • 无(默认情况下，仅显示当前目录内容)

  • Blend 文件 (blend文件的全部内容，仅在链接或附加数据块时可用)

  • 一级

  • 二级

  • 三级

Hint

一次显示几个级别的目录非常方便，例如，可以看到你的全部纹理集合，即使你已经把它们排列在一组很好的目录中，以避免在一个地方有数百个文件。

在 追加/关联 的情况下，显示整个blend文件的内容将允许您在一个操作中关联不同类型的数据块。

Warning

一次显示的级别越多，将需要的时间越多(通常是指数级的，一次显示三个级别可能需要三个数量级的时间完全列出)。

文件过滤¶

标题栏的右边是过滤选项。

第一个 "漏斗"按钮控制是否启用筛选。

文件类型

按分类过滤文件，如文件夹、blend文件、图像等。

数据块类型

在追加或关联时，还可以按数据块类别(如场景、动画、材质等)进行筛选。

Search Ctrl-F

Filter items by name. The wildcard * will match anything, e.g. bl*er will match both blender and blogger. There is always an implicit wildcard at start and end of the search text, so blender will also match test_blender_file.blend. This field can also be used to filter some specific file extension (e.g. .png will list all PNG files).

文件区域¶

文件路径

当前文件夹路径的文本框。 Tab 将自动补全现有路径。 如果键入一个不存在的目录路径，系统将提示您创建该新目录。

文件名

编辑文件名和扩展名的文本字段。保存时，如果背景为红色，则文件夹中已经存在同名文件。 Tab 将自动完成到当前目录中的现有名称。

增量文件名 +, -

在文件名中添加/增加或删除/减少尾随号(例如用于存储文件的不同版本)。

确认 Return

验证操作的主按钮，该按钮定义其名称。双击非目录项将产生同样的效果。

取消 Esc

取消文件选择 (和基础操作)，并关闭文件浏览器。 在 标题栏 使用 返回前一视图 按钮将产生同样的效果。

工具栏¶

左侧区域分为两个区域，顶部是书签，底部是操作面板。

书签¶

上面的列表显示了四种 列表视图 访问某些目录的不同方法。单击其中一个快捷条目将立即导航到该文件夹。

卷

包含所有操作系统定义的可用卷，例如驱动器或网络位置。

系统

包含操作系统定义的通用目录，如主用户文件夹…

收藏夹

包含您希望能够经常访问的文件夹，而不需要在文件浏览器中找到它们。在列表的右边是一些按钮，用于对您的收藏夹进行基本的管理操作，例如添加/删除条目、在列表中向上或向下移动等。

最近打开的文件

包含最近访问的文件夹。右边的 X 按钮允许您完全清除这个列表。

您可以使用设置中 保存 & 加载 选项卡的 最近打开的文件 选项来控制此列表中显示的文件夹数。

操作面板¶

显示调用运算符的选项。 除了下面列出的常见操作之外，许多导入/导出 插件 也将在那里显示它们的选项。

打开，保存，另存为Blender 文件

见 打开 & 保存.

打开，替换，另存为图像

见 支持的图像格式.

从库中链接/附加

见 关联库.

对于普通选项：

相对路径

见 相对路径.

主要区域¶

导航¶

输入目录

在一个目录单击 LMB 进入该目录。

父目录 P

让你到上一级目录。

文件拖拽¶

You can also drag and drop a file or directory from your file manager into the Blender File Browser. This will move it to the directory of the dropped file, and the file will be selected.

选择¶

选择

LMB 和 RMB 点击都有效. 按着 Shift 将扩展选择项目。

选择所有(取消选择) A

切换选择所有文件。

拖拽

按着 LMB 拖动将启动 框选。

方向键¶

也可以通过使用方向键“走步”来选择/取消选择文件：

  • 只使用方向键，将选中所选方向的下一个文件，并取消选择所有其他文件。

  • 在执行此操作时按住 Shift 不会取消选择任何内容，因此它可以扩展到选择内容，并且允许通过导航到已经选择的文件块来取消选择文件(按顺序至少包含两个文件)。

  • 按住 Shift-Ctrl 进一步选择/取消选择中间的所有文件。

如果未选择文件，方向键导航会根据箭头方向选择目录中的第一个或最后一个文件。

如果你选择一个目录并点击 Return ，你将进入该目录(并高亮显示 '父级' .. 条目会将你回到上一个级别)。

文件管理¶

删除文件 Delete, X

删除当前选定的文件。

重命名 F2

修改当前选择的文件或目录的名称。

Warning

小心，没有办法撤销这些操作!

预览¶

文件浏览器处于 缩略图 模式。¶

在缩略图显示模式下，文件浏览器支持多种类型的预览。其中包括：

  • 图像和视频格式

  • 字体

  • Blend文件

  • 内部 数据块

See Blend-files Previews for how to manage Blender data previews.
