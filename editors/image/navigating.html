导航¶

可以通过点击 MMB 并拖动来完成平移。

Zooming can be done by scrolling Wheel up or down. Also, as in the 3D Viewport, you can use NumpadPlus or NumpadMinus to zoom.

小控件¶

在顶部的侧栏旁边，有小控件可以在当没有鼠标滚轮可用时更舒适地进行平移和缩放。

视图菜单¶

区域控制

调整图像编辑器中可见的区域。

自动更新

更新多个区域中的视图。

显示元数据

Displays the metadata if they were set in the render tab's 元数据 panel.

Display Texture Paint UVs

Toggles UVs in Paint Mode.

Zoom In/Out Wheel

Adjusts the zoom level.

分(倍)数变焦缩放

  • 1:8比例变焦缩小 Numpad8

  • 1:4比例变焦缩小 Numpad4

  • 1:2比例变焦缩小 Numpad2

  • 1:1比例变焦缩小 Numpad1

  • 2:1比例变焦放大 Shift-Numpad2

  • 4:1比例变焦放大 Shift-Numpad4

  • 8:1比例变焦放大 Shift-Numpad8

框显全部 Home

将视图居中到整个图像。

框显完全匹配 Shift-Home

使视图适合图像尺寸。

视图中心对齐光标

当 2D 光标可见时，移动视图，使其位于编辑器的中心。

渲染区域 Ctrl-B

请参阅 渲染区域 。

清除渲染区域 Ctrl-Alt-B

请参阅 渲染区域 。

区域

调整图像编辑器所在的 区域 。
