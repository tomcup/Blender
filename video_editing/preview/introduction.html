简介¶

序列编辑器预览用于显示序列编辑器时间轴的渲染结果。这可进一步配置为来自确定的通道，叠加或图像分析器(范围)的输出显示。可以 NumpadPlus 放大视图， NumpadMinus 缩小视图。

标题栏¶

序列编辑器标题栏。¶

视图菜单¶

侧栏 N

显示或隐藏 侧栏。

Toolbar T

Show or hide the Toolbar.

适应窗口预览 Home

Resize the preview so that it fits in the area.

缩放 Shift-B

单击并拖动以绘制矩形并缩放到此矩形大小。

分(倍)数缩放

从1:8到8:1逐步调整预览大小。

Show Frame Overlay

显示 帧叠加，用于将当前帧与参考帧进行比较。

显示安全区

在预览中显示覆盖层，标记出标题安全区域。

显示元数据

在预览区域显示图像元数据。

显示标注

在预览区域中显示 标注 。

序列渲染图像

Render the an image at the current frame.

序列渲染动画

从时间轴的起始帧到结束帧，将预览渲染为一个视频文件或者图像序列。

导出字幕

将用作字幕的 文本片段 导出为 SubRip 文件 (.srt)。导出的文件包含视频序列中的所有文本片段。

Toggle Sequencer/Preview Ctrl-Tab

Switch the editor display type between Sequencer and Preview.

显示模式¶

用于显示当前帧合成结果的不同方面的模式：

图像预览

渲染图像预览.

亮度波形

亮度/对比度分析器.

色度矢量显示器

色调和饱和度。

直方图

RGB直方图分布。

显示通道¶

色彩和Alpha

通过棋盘模式使用透明度来显示预览图像。

色彩

忽视预览图像的透明度(所有透明区域将不显示)。

控制器(Gizmos)¶

在序列编辑器预览区域使用控制器来移动和缩放图像。
